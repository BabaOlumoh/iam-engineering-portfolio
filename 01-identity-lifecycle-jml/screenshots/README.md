## Screenshot Rules

- Redact real names, domains, email addresses, tenant IDs, and IP addresses if using a non-lab environment.
- Use lab users only for portfolio evidence.
- Avoid exposing secrets, passwords, session tokens, or security keys.
- Each screenshot should prove one specific control or outcome.
- Keep filenames consistent with `test-evidence.md`.
