# IAM Engineering Portfolio

This repository contains hands-on Identity and Access Management projects focused on Microsoft Entra ID, Active Directory, hybrid identity, RBAC, identity lifecycle, access governance, automation, and operational runbooks.

The goal of this portfolio is to demonstrate practical IAM skills, including:

- Joiner, Mover, Leaver lifecycle design
- Hybrid identity with Active Directory and Microsoft Entra ID
- RBAC and AGDLP group design
- Conditional Access and Zero Trust policy design
- Privileged Identity Management
- Entitlement Management and access packages
- IAM automation using PowerShell


## Featured Projects

| Project | Skills Demonstrated |
|---|---|
| Identity Lifecycle JML | Joiner, mover, leaver, source of truth, automation |
| Hybrid AD + Entra ID | Entra Connect, AD groups, sync, hybrid identity |
| RBAC AGDLP Model | Role-based access, group nesting, least privilege |
| Conditional Access Lab | MFA, device compliance, risk-based access |
| PIM Lab | Just-in-time admin access, approval, auditing |
| Entitlement Management | Access packages, temporary access, approval workflow |
